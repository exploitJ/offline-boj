
## Features
- Searching with problem id / title
- Preview problems as markdown
- Test solutions with all provided test examples
- keeping track of your own solutions and switching back and forth
- every data is cached to disk, fallback to use cache if offline
- ...and more

## Required Dependencies
- bash
- curl
- jq
- html-xml-utils
- fd
- fzf
- realpath
- pandoc
- chafa
- delta

## Usage
1. clone this repository into a empty directory you'd like to keep your solutions
2. run creath.sh for searching and generating based on template
3. run runner.sh for compiling and validation
